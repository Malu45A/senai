Utilize a Tag header

Usando o Header

O header é utilizado para representar o cabeçalho de um documento ou seção declarado no HTML. Nele podemos inserir elementos da tag h1 a h6, até elentos para representar imagens,parágrafos ou mesmo listas de navegação.

Usando a Tag article

Usando o Artivle

Utilizando o elemento article quando precsamos declarar um contéudo que não precisa de outro para fazer sentido em um documento HTML, por exemplo, um artigo em um blog. É recomendado identificar cada article com um título.

Utilize a Tag section

Usando a Section

Seção 1

O elemento section representa uma seção dentro de um documento e geralmente contém um título, o qual é definido por meio dos elementos h1 e h6. Podemos utilizar o section, por exemplo, para descrever as seções/tópicos de um documento.
